# dapps-browser
DApps browser

[![Build Status](https://travis-ci.org/TrustWallet/dapps-browser.svg?branch=master)](https://travis-ci.org/TrustWallet/dapps-browser)

## Get Started
```
git clone https://github.com/TrustWallet/dapps-browser.git
cd dapps-browser
npm install
npm start
```

## Deploy
`npm run deploy`
